# Ethereum Address Checker

This is a simple hardcoded script to check if an Ethereum address is a wallet or a contract using alchemy.

## Pre-requisites

- Node.js
- Yarn
- Alchemy API key

## Installation

```sh
yarn
```

## Usage

```sh
yarn dev
```

## Build & Run

```sh
yarn build
yarn start
```
